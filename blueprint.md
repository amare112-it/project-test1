
# **Project Blueprint: Lotto Number Generator**

## **1. Overview**

This project is to create a simple and visually appealing Lotto Number Generator. It will provide users with a set of randomly generated numbers for their lottery tickets.

## **2. Features & Design**

### **Core Features**

*   **Number Generation:** Generates a set of 6 unique random numbers between 1 and 45.
*   **User Interaction:** A simple button to trigger the number generation.
*   **Display:** Clearly displays the generated numbers.

### **Design**

*   **Layout:** A clean, single-column layout focused on the core functionality.
*   **Styling:** Modern and engaging design with interactive elements. It will feature a prominent "Generate" button and a clear display for the numbers.
*   **Visuals:** Each number will be displayed in a colored circle for better readability and a more playful look.

## **3. Implementation Plan**

1.  **HTML Structure (`index.html`):**
    *   Create a main container.
    *   Add a header for the site title.
    *   Include a button to generate the numbers.
    *   Set up a container to display the generated lotto numbers.

2.  **CSS Styling (`style.css`):**
    *   Style the main layout and center the content.
    *   Design the "Generate" button with a modern look and hover effects.
    *   Style the display area for the lotto numbers, using colored circles for each number.

3.  **JavaScript Logic (`main.js`):**
    *   Create a function to generate 6 unique random numbers from 1 to 45.
    *   Add an event listener to the "Generate" button to call the number generation function.
    *   Implement the logic to display the generated numbers dynamically in the designated HTML container.
